Working on House Rent Predictor

  • Source code

Building a House Price Predictor using basic ML algorithms on dataset of Boston city. Utilizing the Pickle module for model serialization. Implementing Flask web application for user-friendly access to the predictor. Users input house features, and the Flask app returns the predicted price.

This project demonstrates a seamless blend of Python's Pickle for model persistence, fundamental ML algorithms, and Flask for an intuitive user interface. Perfect for beginners exploring machine learning and web development integration. Dataset available on Kaggle.

  • Used pickle module
  • Used basic ML algorithm
  • Integrating Flask for user-end access
