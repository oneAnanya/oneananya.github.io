Right Rotation Word Problem

Charlie has a magic mirror. The mirror shows right-rotated versions of a given word.

To generate different right-rotations of a word, write the word in a circle in clockwise order, then start reading from any given character in clockwise order till you have covered all the characters.

For example: In the word "sample", if we start with 'p', we get the right rotated word as "plesam". There are six such right rotations of "sample" including itself.

The inputs to the function isSameReflection consists of two strings, word1 and word2. The function returns 1 if word1 and word2 are right rotations of the same word and -1 if they are not. Both word1 and word2 will strictly contain lowercase letters only.

Pseudocode:

Start

Check if lengths of word1 and word2 are equal

If not equal, return -1

Concatenate word1 with itself

Check if concatenated string contains word2

If it contains word2, return 1

Else, return -1

End

Algorithm:

1. Check if the lengths of word1 and word2 are equal.

2. If the lengths are not equal, return -1.

3. Concatenate word1 with itself.

4. Check if the concatenated string contains word2 as a substring.

5. If it does, return 1.

6. Otherwise, return -1.

Java Implementation:


public class MagicMirror 
{
    public static int isSameReflection(String word1, String word2) 
    {
        // If the lengths of the words are different, they can't be rotations of each other
        if (word1.length() != word2.length()) 
        {
            return -1;
        }
        
        // Concatenate word1 with itself
        String concatenated = word1 + word1;
        
        // Check if word2 is a substring of the concatenated word1
        if (concatenated.contains(word2)) 
        {
            return 1;
        } 
        else 
        {
            return -1;
        }
    }

    public static void main(String[] args) 
    {
        // Test cases
        System.out.println(isSameReflection("sample", "plesam")); // Should return 1
        System.out.println(isSameReflection("sample", "maples")); // Should return -1
        System.out.println(isSameReflection("sample", "lesamp")); // Should return 1
        System.out.println(isSameReflection("sample", "samples")); // Should return -1
    }
}
