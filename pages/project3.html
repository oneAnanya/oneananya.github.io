Project Details

🏡 Project Overview

Property Palace is a full-stack MERN real estate platform allowing users to browse, list, and book properties. Firebase secures the platform with user authentication and role-based access control.

📦 Data Handling

  • MongoDB for property data storage.
  • Express.js APIs for CRUD operations.
  • React + Redux for state management.

⚙️ Authentication & Security

  • Firebase Authentication for signup, login, and session management.
  • Role-based access (Admin, User).
  • Protected routes in React with Firebase tokens.

🚀 Deployment

  • Frontend deployed on Vercel.
  • Backend hosted on Render.
  • Connected with Firebase project for auth services.
