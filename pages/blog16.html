Aggregators in SQL

Aggregators in SQL are used to perform calculations on multiple rows and return a single value. Common functions include COUNT, SUM, AVG, MIN, and MAX.

Example

Students Table:

StudentID Name Score
1 Ananya 85
2 Ankita 90
3 Archita 95

COUNT

SQL: SELECT COUNT(StudentID) AS TotalStudents FROM Students;

Output:

TotalStudents
3

SUM

SQL: SELECT SUM(Score) AS TotalScore FROM Students;

Output:

TotalScore
270

AVG

SQL: SELECT AVG(Score) AS AverageScore FROM Students;

Output:

AverageScore
90

MIN

SQL: SELECT MIN(Score) AS MinimumScore FROM Students;

Output:

MinimumScore
85

MAX

SQL: SELECT MAX(Score) AS MaximumScore FROM Students;

Output:

MaximumScore
95
