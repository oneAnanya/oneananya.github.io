Institute Database and SQL Queries

Let's say we have an Institute database with the following tables:

1. Students Table

This table contains basic information about the students.

Student ID Name
101 Ankita
102 Ananya
103 Archita

2. Courses Table

This table lists the courses each student is enrolled in along with the duration.

Student ID Course Duration
101 PGDCA 6 months
102 Full Stack 6 months
103 BI 8 months

3. Student-Hobby Table

This table records the hobbies of each student.

Student ID Hobby
101 Painting
101 Reading
102 Dancing
102 Singing

4. Course-Teacher Table

This table links courses to the teachers who teach them.

Course Teacher
PGDCA Mr. Sharma
Full Stack Mrs. Das
BI Mr. Singh

5. Student-Teacher Table

This table shows which teachers are associated with which students.

Student ID Teacher
101 Mr. Sharma
101 Mrs. Das
102 Mrs. Das
103 Mr. Singh

SQL Queries

Sure! Let's play with SQL queries on the above tables in the "Institute" database.

1. Query to Show All Data from the Students Table

SQL Query:

SELECT * FROM Students;

Output:

Student ID Name
101 Ankita
102 Ananya
103 Archita

2. Query with a HAVING Clause

Let's use the Courses table and count the number of courses each teacher teaches, and then use HAVING to filter results.

SQL Query:

SELECT Teacher, COUNT(Course) AS NumberOfCourses
FROM CourseTeacher
GROUP BY Teacher
HAVING COUNT(Course) > 1;

Output:

Teacher NumberOfCourses
Mrs. Das 2

3. Query to Perform a Table Join

Join the Students and Courses tables to show student names along with the courses they are enrolled in.

SQL Query:

SELECT Students.StudentID, Students.Name, Courses.Course, Courses.Duration
FROM Students
JOIN Courses ON Students.StudentID = Courses.StudentID;

Output:

Student ID Name Course Duration
101 Ankita PGDCA 6 months
102 Ananya Full Stack 6 months
103 Archita BI 8 months

4. Query with Two Satisfying Conditions

We want the students enrolled in the "Full Stack" course and whose duration is "6 months".

SQL Query:

SELECT Students.StudentID, Students.Name, Courses.Course, Courses.Duration
FROM Students
JOIN Courses ON Students.StudentID = Courses.StudentID
WHERE Courses.Course = 'Full Stack' AND Courses.Duration = '6 months';

Output:

Student ID Name Course Duration
102 Ananya Full Stack 6 months

5. SQL Query for 2 Tables

A query to find the names of students along with the names of their teachers.

SQL Query:

SELECT S.StudentID, S.Name, T.Teacher
FROM Students S
JOIN StudentTeacher ST ON S.StudentID = ST.StudentID
JOIN CourseTeacher CT ON ST.Teacher = CT.Teacher;

Output:

Student ID Name Teacher
101 Ankita Mr. Sharma
101 Ankita Mrs. Das
102 Ananya Mrs. Das
103 Archita Mr. Singh
