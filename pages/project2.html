Project Details

üè† Project Overview

Property Price Predictor predicts property prices using supervised learning on a curated Indian cities dataset. The project showcases end-to-end ML lifecycle: data cleaning, feature engineering, model training, evaluation, and deployment.

üßæ Dataset & Preprocessing

  • Source: Kaggle ‚Äî combined and cleaned duplicates, normalized city names.
  • Steps: Impute missing values, encode categorical variables, scale numeric features, create derived features (price per sqft, age buckets).

‚öôÔ∏è Model Training & Evaluation

Used cross-validation and grid/random search for hyperparameter tuning. Final selection based on validation R¬≤ and RMSE.

Model R¬≤ (val) RMSE (val) Notes
Linear Regression 0.82 0.35 Fast, interpretable baseline
Random Forest 0.91 0.22 Strong generalization; served as main model
Gradient Boosting 0.92 0.20 Best accuracy after tuning

üîÅ API & Deployment

  • Flask app exposes /predict (POST) that accepts JSON payload and returns price estimate.
  • Server stack: Gunicorn + Nginx; basic HTTPS using Let's Encrypt (optional).
  • Logging and simple request validation included; model artifacts saved using joblib.

üì¶ How to run locally

  1. Clone the repo: git clone <repo-url>
  2. Create venv and install: python -m venv venv && pip install -r requirements.txt
  3. Run Flask: gunicorn -w 4 app:app or python app.py for dev.
  4. Test: POST /predict with JSON fields: city, area_sqft, bhk, age, amenities.
